3 กฎและเกณฑ์การคำนวณ (Business Rules Engine)

⭐ โหมด Workshop Default

  • จันทร์–พฤหัส: เข้า 08:00 | ออก 17:00 น.
  • ศุกร์ & Pre-Holiday: เข้า 07:00 | ออก 16:00 น.
  • สาย > 1 นาที → ตัดสิทธิ์ค่าข้าว 0฿
  • ตรงเวลา + ทำงานจริง → +25฿/วัน

📋 โหมด Office

  • เข้างาน: ไม่เกิน 09:00 น.
  • ชั่วโมงงาน: ต้องครบ 8 ชั่วโมง
  • มาสาย หรือไม่ครบ 8 ชม. → ตัดสิทธิ์

🗓️ ระบบวันหยุดและ Pre-Holiday Auto-Mapping

ฟีเจอร์รายละเอียด
วันหยุดพรีโหลด18 วันหยุดราชการไทยปี 2026 พร้อมใช้งานทันที
Auto Pre-Holidayระบบหา "วันทำการก่อนหน้า" อัตโนมัติ เช่น หยุดอังคาร → จันทร์ปรับเป็น 07:00 น.
Manual Overrideเพิ่มวัน 07:00 น. แบบ Manual ได้ตามต้องการ
Storageบันทึกผ่าน POST /api/holidays → holidays.json

7 API Endpoints (Local Server Mode)

MethodEndpointหน้าที่
GET /api/holidays ดึงข้อมูลวันหยุดทั้งหมดจาก holidays.json
POST /api/holidays บันทึก/อัปเดตวันหยุดบริษัท

GET /public/* Serve Static Files — HTML, JS, CSS, JSON
