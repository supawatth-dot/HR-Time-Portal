## 🌟 ภาพรวมระบบ (Executive Summary)

ระบบเว็บแอปพลิเคชันสำหรับฝ่ายทรัพยากรบุคคล (HR) และผู้บริหาร ในการประมวลผลข้อมูลการเข้า-ออกงานจากเครื่องรูดบัตร/แตะบัตร เพื่อคำนวณสิทธิ์การเบิก **"ค่าข้าววันละ 25 บาท"** ของพนักงาน Workshop ได้อย่างแม่นยำและรวดเร็ว รองรับข้อมูลมากกว่า 15,000+ รายการ โดยคำนวณเสร็จสิ้นในเวลาไม่ถึง 1 วินาที!

## ⚖️ กฎและเกณฑ์การคำนวณ (Core Business Rules Engine)

### 1. หลักเกณฑ์เวลาเข้างานและค่าข้าว (Working Schedule & Allowance Rule)
- **วันจันทร์ – วันพฤหัสบดี (Mon – Thu):** เวลาเข้างานเป้าหมายคือ **08:00 – 17:00 น.** (ทำงาน 8 ชั่วโมง ไม่รวมพักเที่ยง 1 ชม.)
- **วันศุกร์ และ วันทำการก่อนวันหยุดบริษัท (Fri & Pre-Holiday):** เวลาเข้างานเป้าหมายปรับเป็น **07:00 – 16:00 น.** อัตโนมัติ!
- **เกณฑ์การตัดสินสาย (Late Tolerance):** หากเข้างานหลังเวลาเป้าหมายเกิน 1 นาที (เช่น เข้าหลังเวลา `08:01:00` หรือ `07:01:00` เป็นต้นไป) ระบบจะนับเป็น **"สาย" (Late) และตัดสิทธิ์ค่าข้าว 25 บาทในวันนั้นทันที (0 บาท)**
- **ผู้ที่มาตรงเวลาหรือก่อนเวลา (และมีการทำงานจริง)** จะได้รับ **ค่าข้าววันละ 25 บาท (+25฿)** สะสมเข้าตารางสรุปรายบุคคลเพื่อยื่นเบิกจ่ายเงินเดือน

### 2. โหมดการคำนวณ 2 โหมดอัจฉริยะ (Switchable Modes)
- **⭐ โหมด Workshop มาตรฐาน (Default Mode):** บังคับใช้เกณฑ์ จ-พฤ 08:00 น. และ ศ-ก่อนหยุด 07:00 น. เป็นหลักตามนโยบายบริษัท
- **📋 โหมดพนักงานออฟฟิศ (Office Mode):** สำหรับคิดส่วนงาน Office โดยมีกฎเกณฑ์คือ: เข้างานไม่เกินเวลา 09:00 น. และต้องทำงานให้ครบ 8 ชั่วโมง จึงจะถือว่าได้ค่าข้าว 25 บาท หากมาสายหรือชั่วโมงทำงานไม่ครบ จะถูกตัดสิทธิ์ค่าข้าวทันที

### 3. ระบบจัดการวันหยุดและวันก่อนหยุดอัตโนมัติ (Holiday & Pre-Holiday Auto-Mapping)
- พรีโหลดวันหยุดราชการและบริษัทปี 2026 เรียบร้อยแล้ว (18 วัน)
- เมื่อมีการเพิ่มหรือแก้ไขวันหยุดบริษัทในระบบ จะทำการคำนวณหา **"วันทำการก่อนหน้า" (Pre-Holiday)** ให้อัตโนมัติ (เช่น วันหยุดตรงกับวันอังคาร วันจันทร์ก่อนหน้าจะถูกปรับเวลาเข้างานเป็น 07:00 น. ทันที) พร้อมตัวเลือกเพิ่มวันเข้า 07:00 แบบแมนนวลได้ตามต้องการ

## 🚀 วิธีการเข้าใช้งานและการติดตั้ง (Quick Start & Deployment)

### วิธีที่ 1: เข้าใช้งานออนไลน์ทันที (No-Installation Online Web App)
สามารถเข้าใช้งานระบบได้ทันทีจากทุกอุปกรณ์ผ่านเบราว์เซอร์ โดยไม่ต้องติดตั้งโปรแกรมหรือเซิร์ฟเวอร์ใด ๆ:
👉 **[https://supawatth-dot.github.io/HR-Time-Portal/](https://supawatth-dot.github.io/HR-Time-Portal/)**

*(เมื่อเปิดบนหน้าเว็บออนไลน์ ระบบจะทำการประมวลผลไฟล์ข้อมูลเริ่มต้น 15,750+ รายการด้วย Client-Side SheetJS Engine โดยอัตโนมัติทันที พร้อมรองรับการลากไฟล์ Excel (`.xlsx`, `.csv`) ของเดือนถัดไปมาวางเพื่อคำนวณใหม่ได้บนเบราว์เซอร์ของท่านเลย)*

---

### วิธีที่ 2: ติดตั้งและรันบนเครื่องเซิร์ฟเวอร์บริษัท (Local Backend Server)
หากต้องการรันผ่าน Express API / Node.js Server ภายในบริษัท ให้ทำตามขั้นตอนดังนี้:

1. โคลน Repository หรือดาวน์โหลดโค้ด:
   ```bash
   git clone https://github.com/supawatth-dot/HR-Time-Portal.git
   cd HR-Time-Portal
   ```
2. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```
3. เริ่มต้นรันเซิร์ฟเวอร์:
   ```bash
   npm start
   ```
4. เปิดเว็บเบราว์เซอร์ไปที่:
   👉 **[http://localhost:3000](http://localhost:3000)**
