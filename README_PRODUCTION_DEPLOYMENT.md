  👉 **[http://10.193.0.107:3000](http://10.193.0.107:3000)** *(หรือใช้ชื่อ Hostname ของเครื่องนี้ได้เลย)*

## 🛠️ 2. ชุดเครื่องมือควบคุมระบบ Production (Batch Scripts)
เราได้เตรียมสคริปต์ควบคุมระบบที่ใช้งานง่ายดายในโฟลเดอร์ `c:\Time\HR-Time-Portal\` เพียงดับเบิ้ลคลิก:

1. **🚀 `1. START-PORTAL-BACKGROUND.bat` (เริ่มระบบแบบซ่อนเบื้องหลัง)**
   * ใช้สำหรับเปิดการทำงานของเซิร์ฟเวอร์แบบเบื้องหลัง (Background Service) 
   * ทำงาน 24/7 โดยไม่มีหน้าต่างหน้าจอ Command Prompt ดำๆ เปิดค้างเกะกะหน้าจอบนเครื่อง

2. **🛑 `2. STOP-PORTAL.bat` (ปิดระบบ)**
   * ใช้สำหรับปิดเซิร์ฟเวอร์ (`Node.js`) ที่รันอยู่บนพอร์ต 3000 อย่างปลอดภัย

3. **🔍 `3. CHECK-STATUS.bat` (ตรวจสอบสถานะและ URL)**
   * ใช้ตรวจสอบว่าตอนนี้ระบบยังเปิดทำงานอยู่หรือไม่ พร้อมแสดงหมายเลข IP Address ในวง LAN ล่าสุดสำหรับส่งให้ผู้ใช้งานท่านอื่น

4. **⚙️ `4. INSTALL-AUTO-START-WINDOWS.bat` (ตั้งค่าเปิดอัตโนมัติตอนเปิดเครื่อง)**
   * **แนะนำให้ดับเบิ้ลคลิก 1 ครั้งเพื่อติดตั้ง** 
   * ระบบจะสร้าง VBScript ในโฟลเดอร์ `Windows Startup` ทำให้ทุกครั้งที่มีการเปิดเครื่องเซิร์ฟเวอร์หรือรีสตาร์ทเครื่องใหม่ (เช่น หลังไฟดับ หรือหลัง Windows Update) ระบบ HR-Time Portal จะเปิดตัวเองทำงานเบื้องหลังขึ้นมาทันทีโดยไม่ต้องรอให้มีคนกดเปิดครับ!

## 📋 3. โครงสร้างและการจัดเก็บข้อมูลในระบบจริง (Data Storage & Backup)
* **📂 ไฟล์ตารางกะงานกลางคืน (Master Shift Schedule):** วางไฟล์ตาราง Excel กะกลางคืนประจำเดือนไว้ที่ 📁 **`C:\HR\`** (เช่น `C:\HR\Nigth Shift of Apr 2026.xlsx`) หรือโฟลเดอร์ `c:\Time\HR-Time-Portal\Data\shift\`
* **📅 ข้อมูลวันหยุดบริษัท (Company Holidays):** เก็บถาวรที่ไฟล์ 📁 **`c:\Time\HR-Time-Portal\holidays.json`**
* **📦 ข้อมูลตั้งต้นและไฟล์สำรอง (Backup):** เมื่อกดปุ่ม *"ล้างข้อมูลฐานข้อมูลทั้งหมดเพื่ออัปโหลดใหม่"* ในระบบ ข้อมูลเก่าจะถูกย้ายไปเก็บสำรองอย่างปลอดภัยที่ 📁 **`c:\Time\HR-Time-Portal\Data\backup\`** อัตโนมัติ
